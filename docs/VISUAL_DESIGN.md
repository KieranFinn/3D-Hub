# 3D Hub 视觉优化建议

> 来源: 小K (kie_minimax_bot) | 日期: 2026-03-23

## 核心问题

**5个主要问题:**
1. 左侧大画布太空
2. 右侧面板太窄、像插件栏
3. 颜色只有"深蓝+粉红"，没有高级感
4. 缺少角色创作产品的"舞台感"
5. 信息层级不明显，页面显得廉价

## 推荐风格方向

**暗色未来感 + 轻玻璃拟态 + 粉紫霓虹点缀**

关键词:
- deep navy
- violet glow
- magenta accent
- glass panel
- studio spotlight
- soft shadows
- cyber minimal

## 视觉升级清单

### 1. 左侧画布 → "舞台化"

- [ ] 背景多层渐变（顶部深蓝紫 → 中央聚光 → 边缘更暗）
- [ ] 中心聚焦光/光晕（椭圆高光）
- [ ] 角色展示底座（半透明圆盘 + 霓虹边缘）
- [ ] 背景装饰线条（轻网格/HUD细线/微弱粒子）

### 2. 右侧面板 → "控制舱"

- [ ] 加宽到 30% 比例
- [ ] 玻璃拟态卡片（半透明 + 微弱边框 + 内发光）
- [ ] 分层更明显：一级标题 / 二级卡片 / 三级控件

### 3. 颜色系统

```
背景基底: 深蓝黑/深紫蓝
主强调色: 玫红/电光粉（主按钮/激活段/标题）
辅助色: 青蓝/冰蓝（hover/次级状态/光晕）
```

### 4. 标题区 → 品牌感

- [ ] 大标题 + 副标题（AI Character Creation Studio）
- [ ] 渐变标题字
- [ ] 左侧细亮线
- [ ] 微弱荧光阴影

### 5. 按钮优化

**预设 chips:**
- 默认: 深色描边
- hover: 边框亮一点
- active: 渐变底 + 外发光

**主按钮:**
- 高亮渐变
- hover 上浮
- 轻微 glow

### 6. 滑杆精修

- [ ] 轨道更细
- [ ] 未激活区域更暗
- [ ] 激活区域渐变色
- [ ] thumb 做成发光小圆点
- [ ] hover 时 thumb 放大
- [ ] 数值用 badge 显示: `1.0x`

### 7. 左侧空白 → 创作感信息层

- [ ] 角落弱提示文案: "Drag to rotate" / "Describe your character"
- [ ] 左上角状态条: 当前预设/模式/模型状态
- [ ] 左下角色概要: 风格/气质/输出状态

### 8. 整体布局

**7:3 比例:**
- 左: 70% (画布舞台)
- 右: 30% (控制面板)

---

## 优先改的5个点

1. 左侧加舞台背景和聚光
2. 右侧面板加宽 + 玻璃卡片化
3. 重做标题区，增强品牌感
4. 重做 slider 和 chips 的精致度
5. 建立完整的主色+辅色系统

---

## 目标效果

像:
- AI 创作工具
- 游戏角色编辑器
- 数字工作台
- 有高级感，但不花哨
